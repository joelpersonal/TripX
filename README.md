# TripX - Travel Recommendations

A travel recommendation system that suggests destinations based on your budget, preferences, and travel style.

## Features

- ML-powered destination matching
- Budget-aware recommendations
- Weather and attraction data
- Interactive web interface

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

## How it works

1. Set your travel preferences (budget, duration, style)
2. Get personalized destination recommendations
3. View detailed analysis and itineraries

Built with Python, Streamlit, and various travel APIs.
